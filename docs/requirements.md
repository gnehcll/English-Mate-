# 产品需求文档 (PRD)

## 产品概述

基于 AI 的英语学习网站，帮助用户通过写作和翻译练习提升英语水平。
AI 自动检查语法、优化表达、评分，并生成个性化笔记和练习。

## 目标用户

任何想提升英语写作和翻译能力的学习者。

---

## 功能需求

### 1. 写作与翻译（核心功能）

**1.1 自由写作模式**
- 用户可输入英文文章
- 提供"题目"开关（默认关闭）
  - 打开后显示题目输入框，用户可输入写作主题
  - 题目信息传给 AI 辅助检查（判断内容是否切题）
- 编辑器支持多行文本输入

**1.2 中译英模式**
- 上方输入框：用户输入中文原文
- 下方输入框：用户输入英文翻译
- 两条内容同时提交给 AI 检查（可对照中文原文评估翻译质量）

### 2. AI 检查（核心功能）

点击"检查"按钮后，AI 返回三层结果：

**A. 语法纠错**
- 找出所有语法错误并给出正确写法
- 标注错误类型：时态 / 介词 / 搭配 / 冠词 / 拼写 / 其他
- 使用不同颜色标签区分错误类型

**B. 自然表达优化**
- 标记"语法没错但不够地道"的句子
- 提供母语者更常用的表达方式
- 附带解释说明

**C. 整体评分**
- 按照雅思考试标准打分（0-9 分）
- 给出主要问题总结
- 指出需要重点提升的方向

### 3. 笔记系统

- 从 AI 检查结果中自动提取：错误单词、生疏词组、短语、句子
- 用户可手动选择哪些内容加入笔记
- 笔记卡片展示：原文内容、中文释义、来源文章、创建时间
- 支持按类型筛选：单词 / 词组 / 短语 / 句子
- 支持编辑和删除

### 4. 练习功能

- 根据用户笔记中记录的薄弱点，AI 生成一篇包含这些知识点的中文文章
- 用户在翻译区将中文翻译成英文
- 翻译完成后可再次使用 AI 检查

### 5. 用户系统

- 邮箱注册和登录
- 每个用户的数据隔离

### 6. 设置页面

- 选择 AI 提供商（OpenAI / Claude / DeepSeek / 通义千问等）
- 输入并保存 API Key（加密存储）
- 选择具体模型

- 界面简洁清爽（Notion 风格）
